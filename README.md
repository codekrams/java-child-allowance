# java-child-allowance

Eine Familie  erhält nach folgender einkommensabhängiger Tabelle Kindergeld. Ein Programm ist gesucht, das nach Eingabe der Kinderzahl und des Einkommens das zu zahlende Kindergeld berechnet.  Bitte als Struktogramm und in Java lösen.

Einkommen	unter 45000	ab 45000
für das 1.Kind	    70€	   70€
für das 2.Kind	  130€	   70€
für das 3.Kind	  220€	 140€
ab (!) dem 4.Kind
(und für jedes weitere Kind)	  240€	 140€ 
Mögliche Bildschirmausgabe: 
Ihr Jahreseinkommen in €: 47900 
Anzahl der Kinder: 3 
Ergebnis: 280 € Kindergeld. 
Programmende Kindergeld.
